# Health Assessment System

مشروع مقرر **CS-290** (تطوير تطبيقات الويب) — تطبيق ويب بسيط لحساب مؤشرات الجسم والسعرات الحرارية اليومية.

**المطور:** حسام الحسون  
**الجامعة:** جامعة المستقبل

---

## المميزات

- إدخال البيانات: الاسم، الطول، الوزن، العمر، الجنس، مستوى النشاط
- حساب **BMI** مع تصنيف (نحيف / طبيعي / زائد / سمين)
- تقدير **الوزن المثالي** (نطاق + عرض أكواب مائية تقريبي)
- حساب **السعرات** (خمول / نشاط / رياضة) بمعادلة Mifflin-St Jeor
- عرض السعرات الحرارية مع التصنيف
- طباعة التقرير

## المعادلات

| المقياس | المعادلة |
|---------|----------|
| BMI | الوزن (كغ) / (الطول بالمتر)^2 |
| الماء | الوزن x 0.033 لتر |
| BMR (ذكر) | 10xوزن + 6.25xطول - 5xعمر + 5 |
| BMR (أنثى) | 10xوزن + 6.25xطول - 5xعمر - 161 |
| السعرات | BMR x معامل النشاط |

معاملات النشاط: خامل 1.2، خفيف 1.375، متوسط 1.55، عالي 1.725.
